# 03 · 法度：要素与常数（机器可读）

完整要素清单（缺一不可纳入考量）：
字体、字号、字距、行距、行长、版式、版心、配置、构成、空间、**油墨、支持体（纸张）** ＋ 读者经验与感觉。

| 常数 | 取值/规则 | 落点 |
|---|---|---|
| 单位 | Q=H=0.25mm；字号/行距落整数刻度，必要时精到 0.1Q | engine: size_Q |
| 基本文字 | 先定（字号＋行距），其余由它推导 | engine: base_text |
| 行距 | leading_ratio 默认 1.7–1.9（久读偏大端） | engine: leading_ratio |
| 行长(measure) | 15–45 字；由版心宽÷字号取整 | engine 自动 |
| 字距 | 默认 0（CJK 满行）；非必要不动；判据=灰度非等面积 | — |
| 黑/白 | 着墨与不着墨同等重要；先看白 | measure: density+rivers |
| 灰度 | robust CV<0.28(合格)/<0.20(白井级) | measure |
| 油墨/纸张 | 记录目标油墨浓淡与纸张，纳入决策日志 | log |
