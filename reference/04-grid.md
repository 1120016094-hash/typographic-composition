# 04 · 构造：网格与版心（机器可读）

## 建格五步（engine 已实现 1–4）
1. 定主从（文字主→正文为基本文字；图主→图注为基准）
2. 定基本文字（字号＋行距）
3. 砌"一格"：高=N行＋(N−1)行距；宽=M个全角字密排
4. 留栏距：横向取整数行、纵向取整数字
5. 分栏：4/4 单栏、3/4 / 2/4 双栏、1/4 四栏；栏宽随网格但回到基本字号相称值

## 版心法度（canon）
- 默认 `van_de_graaf`（九分法），即 2:3:4:6（内:上:外:下）传统的构造来源；版心偏上偏内。
- 可选 `explicit` 自定四边（mm）。
- 校验：measure.py 的 canon_match（容差 30%）。

## 反死板
同一套网格要排出变化（非每页一个样）；用"眼"（measure 复核）把僵硬处去掉。
